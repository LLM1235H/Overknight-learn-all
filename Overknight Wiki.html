What is our main goal?

Our Main Goal is to create games which run on all Android Devices no matter the Ram. Our games are made to run smoothly on 500mb Ram And above.

Overknight

Overknight is a mobile game that works on 500mb Ram and above with no lag on the lowest settings.Overknight's development officaly started some were during December 2023 and is set to release late 2024 or early 2025.

Overknight is currently being made in ITsMagic Game Engine but is set to change Game engines further down the road. Overknight is set to get weekly bug fixes and monthly content updates after it releases.
